# Внешние маршруты OSPF и их приоритет

OSPF поддерживает импорт маршрутов из других автономных систем через **ASBR (Autonomous System Boundary Router)**.

## Типы внешних маршрутов

| Тип | Название | Формула метрики | Используется когда |
|------|-----------|------------------|--------------------|
| **E1** | External Type 1 | Включает стоимость до ASBR + внешнюю стоимость | Когда важен точный путь внутри OSPF |
| **E2** | External Type 2 | Использует только внешнюю стоимость (внутренняя не учитывается) | Когда важна только внешняя метрика |

## Приоритет выбора маршрута

1. **Intra-area (внутри области)**  
2. **Inter-area (межобластной)**  
3. **Type 1 External (E1)**  
4. **Type 2 External (E2)**  
5. **NSSA External (Type 7)**
