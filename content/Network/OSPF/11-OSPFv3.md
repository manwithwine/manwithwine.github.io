# OSPFv3 (для IPv6)

OSPFv3 — обновленная версия протокола для работы с IPv6.

## Отличия от OSPFv2
- Работает поверх **IPv6** (протокол №89 тот же).
- Использует **Link-Local адреса** для соседства.
- Не передает IP-префиксы в LSA — они описываются отдельно.
- Поддерживает **аутентификацию через IPv6 IPsec**.
- Появились новые типы LSA (Link-LSA, Intra-Area-Prefix-LSA).
