#summary
## $$Summary$$
- On an MSTP network, one or more VLANs can be mapped to an MSTI, and MSTP calculates the spanning tree based on the MSTI. The MSTI is an instance-based spanning tree. MSTP maintains a spanning tree for each independent MSTI. The VLANs mapped to the same MSTI share the same spanning tree
- MSTP can implement the following functions on the Ethernet:
	- Forms multiple loop-free trees to prevent broadcast storms and implement redundancy.
	- Multiple spanning trees perform load balancing and transmit traffic in different VLANs along different paths
